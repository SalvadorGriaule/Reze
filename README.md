 Description du projet
SkillShare Mini est une micro-plateforme d’échange de
compétences.
Chaque utilisateur peut :
● indiquer les compétences qu’il peut enseigner (ex :
HTML, Excel, guitare),
● indiquer les compétences qu’il souhaite apprendre,
● consulter les profils des autres utilisateurs
